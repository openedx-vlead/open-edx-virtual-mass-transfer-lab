LABVIEW CALCULATIONS :

(Note : In the snapshots all the boxes with a green background designates inputs/measurements ,with a blue background designates virtual data and with a pink background desinates user's calculated data.)

STEP 1 - It includes  the calibration part where the user will get the virtual value for refractive index on inserting the mole fraction data and pressing enter.10 such different values are to be inserted in order to complete the step.

 

STEP 2 - In this step the user will get to see all his entered mole fraction data and the virtual RI data.Then he should plot a graph for further calculations.

 

STEP 3 - This is the virtual experiment part where the inputs are reflux and feed composition and the virtual output are RI value for both partial and total reflux.

 

STEP 4 - In this step the user has to enter his calculated values of effeciency and number of equilibrium stages based on the virtual data obtained from step 3.

 

STEP 5 - This is a simple comparison step between the user's calculated data and the virtual data.

 

STEP 6 - This is the concluding step mentioning the marks the user gets on his performance.
